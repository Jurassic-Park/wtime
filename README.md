## golnag time handle

###How to use ?
``
go install https://github.com/Jurassic-Park/wtime.git
``

``
t := newWTime(...)  
``
#### time.Time
``
t.ToTimeObj()
``
#### 2020-10-10 00:00
``
t.ToTimeStr()
``
#### 2020-10-10
``
t.ToDateStr()
``
#### timestamp
``
t.ToTimestamp()
``
#### millisecond
``
t.ToMillisecond()
``
